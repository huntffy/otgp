# Open Telecom GIS Planner (OTGP)

Mission:

Build the world's best open-source GIS platform for telecommunications engineering.

The platform should replace multiple disconnected engineering tools such as spreadsheets, Google Earth workflows, CAD drawings, simple RF calculators, and proprietary planning utilities with one modern, extensible platform.

This is NOT just another calculator application.

It is an engineering platform.

----------------------------------------------------
Core Vision
----------------------------------------------------

Create an open-source ecosystem where telecommunications engineers can:

• Plan RF networks
• Design Fiber networks
• Manage towers
• Calculate microwave links
• Visualize GIS data
• Generate reports
• Extend the system using plugins
• Integrate with external systems through APIs

The architecture must be modular, scalable, maintainable, and suitable for thousands of future contributors.
